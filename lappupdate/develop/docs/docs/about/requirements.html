The requirements for this project were:

  • lappdeploy must run on Microsoft Windows (7 minimum) without prerequisites (i.e. no agent must be prior installed). Theses implies that theses modules must be written in Command shell or Windows Script Host.
  • lappdeploy``lappdeploy must run from a network share (aka from a UNC path) or a removable disk (CD or DVD, USB stick...).
  • lappdeploy``lappdeploy must work with any type of installation package (MSI package, EXE package or a classic distribution with files and a setup.exe).
  • lappdeploy``lappdeploy must have pre install and post installation hook to customize the start menu or install additional packs (e.g. Firefox extension, VirtualBox Extension Pack, Tortoise Language Pack...)
