support – Supporting Modules Package¶

This package is the place where the supporting module are defined. Theses modules provide support for the lapptrack and the product handlers grouped in the cots package.

  • support.report – Reporting Manager
  • support.pad – Lightweight Portable Application Description Support
  • support.progressbar – Progress Bar Widgets
